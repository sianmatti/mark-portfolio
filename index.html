Case Study 01

Splash Park Bacolod

Attraction / Event Promotion
Goal

Increase event turnout while improving acquisition efficiency.

Challenge

The campaign needed stronger audience structure, better promo testing, and better tracking visibility to improve turnout without pushing costs too high.

What I did
  • Structured audience segmentation to improve targeting efficiency
  • Built a prospecting vs retargeting system to reduce overlap
  • Created 7-day, 14-day, and 30-day remarketing windows
  • Supported promo testing to identify stronger-performing campaign angles
  • Implemented GA4 event tracking and dashboard reporting
  • Used reporting insights to guide ongoing optimization decisions
Result

Increased event turnout by approximately 35% while reducing CPA by around 25%. The segmentation and remarketing structure helped improve turnout while lowering acquisition cost.

Tools used

Meta Ads Manager • GA4 • Reporting Dashboards

Key takeaway

This project reflects my ability to improve both scale and efficiency by combining audience strategy, retargeting logic, and performance tracking.
